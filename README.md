# mylife-home-kmodules
MyLife Home Linux kernel modules

## AC

Provide `ac_zc`, `ac_dimmer` and `ac_button`

Modules built from rpi-alpine-build

Development:
```shell
make -C dev-tools/Makefile.ac <rule>
```

## PWM

Provide `dma_pwm`

Modules built from rpi-alpine-build

Development:
```shell
make -C dev-tools/Makefile.pwm <rule>
```
